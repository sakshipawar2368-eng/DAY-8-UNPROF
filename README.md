# DAY-8-UNPROF
*# 🌍 COVID-19 Data Analysis Dashboard (Day 8 Final Project)

## 📌 Project Overview

This project demonstrates an end-to-end data pipeline using Python. It fetches real-time COVID-19 data from a public API, cleans and analyzes the data, visualizes the results using charts, generates a PDF report, and saves the processed data in CSV and JSON formats.

---

## 🚀 Features

- Fetch data from a Public API
- Clean and process data using Pandas
- Analyze COVID-19 statistics
- Create Bar, Line, and Pie charts using Matplotlib
- Generate a PDF report
- Save processed data as CSV and JSON

---

## 🛠 Technologies Used

- Python
- Requests
- Pandas
- Matplotlib
- ReportLab
- Jupyter Notebook (Anaconda)

---

## 📂 Files Generated

- `processed_covid_data.csv`
- `processed_covid_data.json`
- `covid_report.pdf`

---

## ▶️ Setup & Run

1. Install required libraries:

   ```bash
   pip install requests pandas matplotlib reportlab
   ```

2. Open `covid_data_pipeline.ipynb` in Jupyter Notebook.

3. Run all cells.

4. View the generated charts and output files.
